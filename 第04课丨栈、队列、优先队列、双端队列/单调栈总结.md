# 单调栈总结

## 什么是单调栈

单调栈是一种基于栈的数据结构，通过维护区间[0, r]数据的单调性，将时间复杂度从O($n^2$)降为O(n)。

## 什么时候使用单调栈

1.数据存在单调性

- 寻找下一个更大(更小)的数
- 确定区间最值
- 以该元素为最值的最长区间

2.从时间复杂度角度，需要从O($n^2$)降为O(n)

## 代码模板

单调递增栈
```
stack<int> st;
for(int i = 0; i < nums.size(); i++)
{
    while(!st.empty() && nums[i] < st.top() )
    {
        /**code**/
        st.pop();
    }
    st.push(nums[i]);
}
```
